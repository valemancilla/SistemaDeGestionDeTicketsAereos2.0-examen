#  PROMPT ENGINEERING

## Sistema de Gestión de Tiquetes Aéreos 

------

##  Prompt 1 — Comprensión arquitectónica del sistema

Quiero que asumas el rol de docente universitario especializado en desarrollo de software utilizando C# orientado a aplicaciones de consola. Actualmente estoy desarrollando un Sistema de Gestión de Tiquetes Aéreos como proyecto académico, cuyo objetivo principal es permitir el registro de vuelos, gestión de pasajeros, creación de reservas y consulta de información mediante interacción directa desde consola.

Durante el desarrollo he logrado implementar funcionalidades individuales, pero aún no comprendo cómo visualizar el sistema como una estructura completa organizada profesionalmente. Mi dificultad principal radica en entender cómo dividir correctamente el proyecto para evitar que todo el código quede centralizado en el archivo principal.

Voy a describirte los módulos que estoy creando dentro del sistema y necesito que construyas una explicación progresiva donde relaciones conceptos de arquitectura básica, separación de responsabilidades y organización de clases, integrando teoría con ejemplos prácticos en C# que me permitan comprender cómo debería estructurarse un proyecto real incluso siendo una aplicación de consola.

El sistema incluye conceptos como pasajeros, vuelos, aerolíneas y reservas, los cuales interactúan entre sí dentro del flujo del programa. Mi dificultad aparece al momento de decidir qué información pertenece a cada clase y qué comportamiento debería encapsularse dentro de ellas.

Te compartiré cómo imagino el funcionamiento del sistema y necesito que desarrolles una explicación pedagógica basada en un caso de estudio real, mostrando cómo transformar conceptos del negocio en objetos bien diseñados, justificando decisiones y acompañando la explicación con ejemplos codificables que conecten directamente con la lógica del sistema.

##  Prompt 3 — Flujo completo del programa en consola

Quiero que adoptes el rol de instructor especializado en desarrollo de aplicaciones interactivas de consola usando C#. Mi sistema funciona mediante menús que permiten al usuario navegar entre distintas opciones como registrar vuelos, crear reservas y consultar información almacenada.

Voy a explicarte cómo funciona actualmente mi menú principal y necesito que desarrolles una explicación detallada donde analices el flujo lógico del programa desde su inicio hasta su finalización, proponiendo una organización clara mediante ciclos, métodos y control de estados, acompañando cada explicación con fragmentos de código que permitan visualizar la solución.

Te describiré las operaciones que realizo actualmente y necesito que construyas una explicación que combine fundamentos teóricos sobre colecciones con ejemplos prácticos aplicados directamente al caso del sistema aéreo, permitiéndome entender cuándo recorrer listas, cuándo filtrar datos y cómo optimizar estas operaciones.
